# minish
The mini shell written in blazing fast C 

Entirely minimal, yet strangely POSIX-compliant. 
Maybe you'll like it, maybe you won't.

To compile, you'll need GNU Readline (already not minimal lol)

On Ubuntu and derivatives including Mint:

```sudo apt install libreadline-dev```

then---

```gcc -o minish minish.c -lreadline```
